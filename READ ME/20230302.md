# get vs indexOf
- get(int index)          
Array의 index번호에 위치한 배열값을 반환해줌
- indexOf(Generic generic)            
Array에 내가 원하는 값이 있는지 확인해줌         
(문자열 중 해당문자 위치 찾기 indexOf() - 특정 문자나 문자열에서 해당하는 문자의 인덱스 값을 반환하고 찾지 못했을 경우 '-1'을 반환하는 메소드 입니다.)
